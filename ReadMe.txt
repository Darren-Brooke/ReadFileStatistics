Elcom Programming Challenge

The solution to the programming challenge is a Python based script that utilises the Command Line Prmopt.

Installation Instructions:

    The script requires Python to be installed. Visit https://www.python.org/downloads/ and follow the instructions as shown

Running the Program:
    On the initial program start it will open a file browser
    Navigate to and select the desired text file
    The word count, line count and mean word count will be displayed in the python command line. 
    Press enter to close the program

Programming Exercise:

We would like you to write a program for generating statistics from a text file. Your program should accept a path to a file as input and produce the following statistics:

Word count
Line count
Mean word length
